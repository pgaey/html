jQuery 개요

jQuery란?

기존에 자바스크립트 언어만으로 구현하기 복잡했던 구문작성들을
간소화하기 위해서 "존 레식"에 의해 개발된
JavaScript기반의 라이브러리(유용한 기능들의 모음집)
즉, JavaScript언어로 유용한 함수나 내용들이 이미 만들어져있는 모음집

- jQuery의 장점
1) DOM요소와 관련된 스크립트를 보다 쉽게 구현할 수 있다.
2) AJAX(비동기식통식), 이벤트 처리 등 폭 넓게 지원한다.
3) jQuery와 관련된 확장플러그인, 오픈 API등을 지원한다. => 적은 양의 코드로 풍부한 기능을 제공한다 !

* jQuery 라이브러리 연결 방법

head태그 내부에 script태그로 jQuery관련 .js파일 가져오기
오프라인 / 온라인 방식이 있음
- jQuery 공식 사이트 : jQuery 사이트

1. jQuery 라이브러리 파일을 직접 다운로드 받은 후 경로 지정(오프라인방식)
- .js파일을 다운로드 받아 저장한 후 저장된 파일의 경로를 지정한다.

2. CDM(Content Delivery Netword)을 통한 연결(온라인 방식)
- 라이브러리를 제공하는 사이트의 url을 제시해서 파일 경로를 지정하는 방식


* jQuery 구문 작성하기

우리는 기본적으로 jQuery구문으로 DOM요소를 다룰 것이기 때문에
jQuery명령은 문서상에서 모든 요소들이 전부 다 로드된 후에 실행되어야 함 => window.onload 


        - 순수 자바스크립트에서 해당 내용
        [ 표현법 ]
        window.onload = function(){
            실행내용 ;
        }

        onload 함수 : 페이지가 다 로딩되면 자동으로 딱 한 번 실행되는 함수
        해당 HTML문서의 요소들도 다 로딩되고, 연동되어있는 외부 문서들도 전부 다 로딩된 후에
        실행되는 특징을 갖는다.
        window.onload는 해당 HTML문서에서 딱 한 번만 수행 가능하다.
        
    
        - jQuery에서는 해당 내용을 3가지 구문으로 제공 
        [ 표현법 ]
        1)
        jQuery(document).ready(function(){
            실행내용;
        })

        2)
        $(document).ready(function(){
            실행내용;
        })

        $ : jQuery 키워드를 나타냄

        jQuery에서 해당 function은 HTML문서 안에 있는 DOM요소들만 로드되면 바로 실행됨
        여러번 기술 가능, 횟수 제한 없음!!

        3)
        $(function(){
            실행내용;
        })

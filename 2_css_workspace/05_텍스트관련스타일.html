텍스트 관련 스타일

        [ 표현법 ]
        선택자{
            color : 값;
        }

text-decoration : 텍스트에 줄을 긋거나 지울 때 사용 

        [ 표현법 ]
        선택자 {
            text-decoration : none / underline / overline / line-through;
        }
  • unserline : 텍스트 아래에 선 긋기
  • overline : 텍스트 위에 선 긋기
  • line-through : 텍스트 중간에 선 긋기

목록 관련 스타일(list~)

list-style-type : 불릿 기호를 변경할 때 사용

        [ 표현법 ]
        * 순서 없는 목록(ul)
        선택자 {
            list-style-type : disc / circle / square / none;
        }

        * 순서 있는 목록(ol)
        선택자 {
            list-style-type : decimal
            list-style-type : lower-alpha / upper-alpha => 알파벳
            list-style-type : lower-roman / upper-roman => 로마숫자
        }
